Privacy Policy for Graceful Quotes

Effective date: October 2025

Graceful Quotes is a Chrome extension that replaces your New Tab page with inspirational quotes and backgrounds. It does not collect, store, transmit, or share any personal or identifying information.

Data Storage

All user preferences (such as selected background, favorites, or theme settings) are stored locally on your device using Chrome’s storage API. This data never leaves your browser and is never sent to any server.

Network Access

The extension may optionally fetch static JSON and image assets (quotes, portraits, or background lists) from the developer’s GitHub Pages:
https://pradyumnasagar.github.io/graceful-quotes-data/. These assets contain only text or image filenames—no executable code or personal information.

Third Parties

Graceful Quotes does not use analytics, advertising, or tracking services. No user data is shared with third parties.
